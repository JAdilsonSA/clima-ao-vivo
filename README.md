# 🌦️ Clima ao Vivo

Projeto desenvolvido para consultar informações meteorológicas em tempo real utilizando a API Open-Meteo.

## 📌 Funcionalidades

* Busca do clima por cidade.
* Exibição da temperatura atual.
* Exibição da velocidade do vento.
* Interface responsiva.
* Consumo de API pública utilizando Fetch API.

## 🛠️ Tecnologias Utilizadas

* HTML5
* JavaScript (ES6+)
* Tailwind CSS
* Fetch API
* Open-Meteo API

## 🌐 APIs Utilizadas

* Geocoding API (Open-Meteo)
* Forecast API (Open-Meteo)

## ⚠️ Limitações

* O sistema depende de conexão com a internet.
* Caso a cidade não seja encontrada, será exibida uma mensagem de erro.
* A aplicação exibe apenas informações climáticas atuais.
